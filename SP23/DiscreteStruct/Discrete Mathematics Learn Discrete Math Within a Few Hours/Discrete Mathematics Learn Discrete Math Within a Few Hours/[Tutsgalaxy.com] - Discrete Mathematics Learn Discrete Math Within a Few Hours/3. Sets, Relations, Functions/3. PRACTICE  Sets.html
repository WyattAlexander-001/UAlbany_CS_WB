Here are a couple of practice questions, You should be able to figure them out. If you get stuck, as allways, post a question :)

Basic:

When A = {a, b, c, d, e} and B = {b, c, e, f, g} What are the results of these operations?

  1. A ⊆ B
  2. |A| = |B|
  3. A∪B
  4. A∩B
  5. A\B
  6. B\A

Advanced:

When A = {1, 2, 3, 4}, B = {1, 2, 7, 8, 9} and C = {5, 6, 7} What are the results of these operations?

  1.  |A| + |B| + |C|
  2. (A∩B) \ C
  3. (A∪B) \ C
  4. C \ (A∪B)
  5. (A∩B) ∩ C

Hard:

Are statements bellow true for every A, B and C?

Is A \ (B ∪ C)  same as (A \ B) ∩ (A \ C) ?

Is (A ∪ B) \ C  same as (A \ C) ∪ (B \ C)  ?
